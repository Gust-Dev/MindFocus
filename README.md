
# MindFocus 🍅

> Mantenha o foco, gerencie seu tempo e aumente sua produtividade.

O **MindFocus** é um aplicativo de Pomodoro simples e minimalista. Desenvolvido para ajudar você a manter a concentração nas suas tarefas diárias, alternando blocos de trabalho profundo com pausas estratégicas, sem distrações desnecessárias.


  Pomodoro Study App 

  This is a code bundle for Pomodoro Study App. 


  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
